# 🥘 K-던전밥: 이명 어르신의 모험 (K-Dungeon Cooking)

잠실 석촌호수 지하에 열린 미궁! 가족들이 돌아올 땅을 지키기 위한 '이명' 어르신의 고군분투 액션 게임입니다.

## 🎮 게임 플레이하기
👉 [게임 바로가기 링크](https://[본인아이디].github.io/k-dungeon-cooking/) 
*(위 링크를 본인의 실제 주소로 수정해주세요)*

## 🕹️ 조작 방법
- **이동:** 방향키 (←, →)
- **점프:** 위쪽 화살표 (↑)
- **공격:** Z 키 (몬스터를 잡아 재료를 얻으세요!)
- **요리:** C 키 (버섯 3개를 모아 요리하세요!)

## 🗺️ 스토리
가족들을 유학 보내고 홀로 잠실을 지키는 고물상 주인 이명.
어느 날 석촌호수 아래에서 몬스터들이 기어 나오기 시작하는데...
"내 집 밑에서 썩은 냄새 풍기는 건 못 참지! 다 요리해버리겠다!"
